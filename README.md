# elasticsearch-kibana

以下のコマンドでelasticsearchとkibanaが立ち上がる
```
docker-compose up -d
```

立ち上がったら 、、 

## elastic search へのアクセス
http://localhost:9200/   

## kibana へのアクセス
http://localhost:5601/
